\# Free Fire Aimbot Controller



🎯 Floating overlay aimbot for Free Fire (Android)



\## Quick Install

1\. Download ZIP

2\. Termux: `buildozer android debug`

3\. Install APK → Enable overlay → Play!



\## Features

\- Headshot lock

\- FOV control (100-600px)

\- Smoothness slider

\- Real-time stats
